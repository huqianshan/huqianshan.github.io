3.2 Cookieе’Ңtimeout

Cookie зҡ„иҝ”еӣһеҜ№иұЎдёә RequestsCookieJarпјҢе®ғзҡ„иЎҢдёәе’Ңеӯ—е…ёзұ»дјјпјҢдҪҶжҺҘеҸЈжӣҙдёәе®Ңж•ҙпјҢйҖӮеҗҲи·ЁеҹҹеҗҚи·Ёи·Ҝеҫ„дҪҝз”ЁгҖӮдҪ иҝҳеҸҜд»ҘжҠҠ Cookie Jar дј еҲ° Requests дёӯ.

д№ҹеҸҜд»ҘзӣҙжҺҘз”ҹжҲҗеӯ—е…ёdict(status='value') дј йҖ’з»ҷrequests

иҜҙжҳҺпјҡ requests еңЁз»ҸиҝҮд»Ҙ timeout еҸӮж•°и®ҫе®ҡзҡ„з§’ж•°ж—¶й—ҙд№ӢеҗҺеҒңжӯўзӯүеҫ…е“Қеә”гҖӮеҹәжң¬дёҠжүҖжңүзҡ„з”ҹдә§д»Јз ҒйғҪеә”иҜҘдҪҝз”ЁиҝҷдёҖеҸӮж•°гҖӮеҰӮжһңдёҚдҪҝз”ЁпјҢдҪ зҡ„зЁӢеәҸеҸҜиғҪдјҡж°ёиҝңеӨұеҺ»е“Қеә”пјҡ

timeout д»…еҜ№иҝһжҺҘиҝҮзЁӢжңүж•ҲпјҢдёҺе“Қеә”дҪ“зҡ„дёӢиҪҪж— е…ігҖӮ timeout е№¶дёҚжҳҜж•ҙдёӘдёӢиҪҪе“Қеә”зҡ„ж—¶й—ҙйҷҗеҲ¶пјҢиҖҢжҳҜеҰӮжһңжңҚеҠЎеҷЁеңЁ timeout з§’еҶ…жІЎжңүеә”зӯ”пјҢе°Ҷдјҡеј•еҸ‘дёҖдёӘејӮеёёпјҲжӣҙзІҫзЎ®ең°иҜҙпјҢжҳҜеңЁ timeout з§’еҶ…жІЎжңүд»ҺеҹәзЎҖеҘ—жҺҘеӯ—дёҠжҺҘж”¶еҲ°д»»дҪ•еӯ—иҠӮзҡ„ж•°жҚ®ж—¶пјүIf no timeout is specified explicitly, requests do not time out.
