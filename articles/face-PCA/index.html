дёҖгҖҒе®һйӘҢз»јиҝ°

дәәи„ёж•°жҚ®йӣҶдёӯпјҢе…ұжңү16дёӘдәәпјҢжҜҸдәәиҮіе°‘жңүдә”е№…еӣҫеғҸпјҢиҰҒжұӮйҖүеҸ– 5 е№…еӣҫеғҸпјҢе…¶дёӯеӣӣе№…еӣҫдҪңдёәи®ӯз»ғйӣҶпјҢеү©дёӢдёҖеүҜеӣҫдҪңдёәжөӢиҜ•йӣҶгҖӮиҰҒжұӮеҲ©з”Ёи®ӯз»ғйӣҶи®ӯз»ғжЁЎеһӢпјҢд»ҺжөӢиҜ•йӣҶд»»йҖүдёҖе№…еӣҫеғҸпјҢиғҪеӨҹиҜҶеҲ«д»–зҡ„иә«д»ҪгҖӮ

жң¬ж–ҮжҖқи·ҜеҰӮдёӢпјҡйҰ–е…ҲеҜ№еӣҫеғҸиҝӣиЎҢйў„еӨ„зҗҶпјҢеҲ©з”ЁжңҖиҝ‘йӮ»з®—жі•йҮҚйҮҮж ·и§„ж•ҙдёә 128*128зҡ„зҒ°еәҰеӣҫеғҸпјҢе№¶еҜ№дә®еәҰејӮеёёзҡ„еӣҫеғҸиҝӣиЎҢи°ғж•ҙгҖӮз„¶еҗҺеҲ©з”ЁGaborж»ӨжіўеҷЁеҜ№зҒ°еәҰеӣҫеғҸиҝӣиЎҢдә”е°әеәҰпјҢе…«дёӘж–№еҗ‘зҡ„зү№еҫҒжҸҗеҸ–гҖӮжҺҘзқҖеҲ©з”ЁPCAз®—жі•еҜ№еӣҫеғҸж•°жҚ®иҝӣиЎҢеҺӢзј©пјҢйҖүеҸ–иҙЎзҢ®зҺҮеүҚ95%зҡ„зү№еҫҒеҗ‘йҮҸдҪңдёәж–°зҡ„еҹәгҖӮжңҖеҗҺеҲҶзұ»з®—жі•йҮҮеҸ–зәҝжҖ§еӣһеҪ’зҡ„еҲҶзұ»еҷЁпјҢйў„жөӢжөӢиҜ•йӣҶдёӯеӣҫеғҸжүҖеұһзұ»еҲ«пјҢе№¶еҫ—еҲ°иҜҶеҲ«зҺҮгҖӮдёәдәҶжҺўи®ЁдёҚеҗҢGaborж»ӨжіўеҷЁзҡ„еҸӮж•°еҜ№иҜҶеҲ«з»“жһңзҡ„еҪұе“ҚпјҢйҖүеҸ–дәҶеӨҡз»„еҸӮж•°иҝӣиЎҢеҸӮж•°ж•Ҹж„ҹжҖ§еҲҶжһҗгҖӮ

дәҢгҖҒе®һйӘҢеҺҹзҗҶ

1.Gaborзү№еҫҒ

GaborеҸҳжҚўеұһдәҺеҠ зӘ—еӮ…з«ӢеҸ¶еҸҳжҚўпјҢGaborеҮҪж•°еҸҜд»ҘеңЁйў‘еҹҹдёҚеҗҢе°әеәҰгҖҒдёҚеҗҢж–№еҗ‘дёҠжҸҗеҸ–зӣёе…ізҡ„зү№еҫҒгҖӮгҖҗ1гҖ‘Gaborе°ҸжіўдёҺдәәзұ»и§Ҷи§үзі»з»ҹдёӯз®ҖеҚ•з»Ҷиғһзҡ„и§Ҷи§үеҲәжҝҖе“Қеә”йқһеёёзӣёдјјгҖӮе®ғеңЁжҸҗеҸ–зӣ®ж Үзҡ„еұҖйғЁз©әй—ҙе’Ңйў‘зҺҮеҹҹдҝЎжҒҜж–№йқўе…·жңүиүҜеҘҪзҡ„зү№жҖ§гҖӮGaborе°ҸжіўеҜ№дәҺеӣҫеғҸзҡ„иҫ№зјҳж•Ҹж„ҹпјҢиғҪеӨҹжҸҗдҫӣиүҜеҘҪзҡ„ж–№еҗ‘йҖүжӢ©е’Ңе°әеәҰйҖүжӢ©зү№жҖ§пјҢиҖҢдё”еҜ№дәҺе…үз…§еҸҳеҢ–дёҚж•Ҹж„ҹ,иғҪеӨҹжҸҗдҫӣеҜ№е…үз…§еҸҳеҢ–иүҜеҘҪзҡ„йҖӮеә”жҖ§гҖӮдёҠиҝ°зү№зӮ№дҪҝGaborе°Ҹжіўиў«е№ҝжіӣеә”з”ЁдәҺи§Ҷи§үдҝЎжҒҜзҗҶи§ЈгҖӮ
йҖҡиҝҮйў‘зҺҮеҸӮж•°е’Ңй«ҳж–ҜеҮҪж•°еҸӮж•°зҡ„йҖүеҸ–пјҢGaborеҸҳжҚўеҸҜд»ҘйҖүеҸ–еҫҲеӨҡзә№зҗҶзү№еҫҒпјҢдҪҶжҳҜGaborжҳҜйқһжӯЈдәӨзҡ„пјҢдёҚеҗҢзү№еҫҒеҲҶйҮҸд№Ӣй—ҙжңүеҶ—дҪҷгҖӮ GaborжҳҜжңүй«ҳж–Ҝж ёеҮҪж•°дёҺеӨҚжӯЈејҰеҮҪж•°и°ғеҲ¶иҖҢжҲҗпјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

еӣҫ(a)дёәеҒҸз§»xиҪҙ30В°зҡ„жӯЈејҰеҮҪж•°пјҢеӣҫ(b)дёәй«ҳж–Ҝж ёпјҢеӣҫ(c)дёәеҜ№еә”зҡ„Gabor filterгҖӮеҸҜд»ҘзңӢеҮәжӯЈејҰеҮҪж•°жҳҜеҰӮдҪ•еңЁз©әй—ҙдёҠе…·жңүеұҖйғЁжҖ§зҡ„гҖӮ

2.PCA

PCAзҡ„еҺҹзҗҶе°ұжҳҜе°ҶеҺҹжқҘзҡ„ж ·жң¬ж•°жҚ®жҠ•еҪұеҲ°дёҖдёӘж–°зҡ„з©әй—ҙдёӯпјҢзӣёеҪ“дәҺжҲ‘д»¬еңЁзҹ©йҳөеҲҶжһҗйҮҢйқўеӯҰд№ зҡ„е°ҶдёҖз»„зҹ©йҳөжҳ е°„еҲ°еҸҰеӨ–зҡ„еқҗж Үзі»дёӢгҖӮйҖҡиҝҮдёҖдёӘиҪ¬жҚўеқҗж ҮпјҢд№ҹеҸҜд»ҘзҗҶи§ЈжҲҗжҠҠдёҖз»„еқҗж ҮиҪ¬жҚўеҲ°еҸҰеӨ–дёҖз»„еқҗж Үзі»дёӢпјҢдҪҶжҳҜеңЁж–°зҡ„еқҗж Үзі»дёӢпјҢиЎЁзӨәеҺҹжқҘзҡ„еҺҹжң¬дёҚйңҖиҰҒйӮЈд№ҲеӨҡзҡ„еҸҳйҮҸпјҢеҸӘйңҖиҰҒеҺҹжқҘж ·жң¬зҡ„жңҖеӨ§зҡ„дёҖдёӘзәҝжҖ§ж— е…із»„зҡ„зү№еҫҒеҖјеҜ№еә”зҡ„з©әй—ҙзҡ„еқҗж ҮеҚіеҸҜгҖӮ

зәҝжҖ§еӯҗз©әй—ҙжҖқжғіпјҡдёҖеүҜдәәи„ёеӣҫеғҸеҸҜз”ұжң¬зұ»дәәи„ёеӣҫеғҸзәҝжҖ§иЎЁзӨәгҖӮ

дёүгҖҒе®һйӘҢжӯҘйӘӨ

1. йў„еӨ„зҗҶ

вҖӢ з”ұдәҺеӣҫеғҸеҗ„ејӮпјҢйў„еӨ„зҗҶжҳҜдәәи„ёиҜҶеҲ«иҝҮзЁӢдёӯзҡ„дёҖдёӘйҮҚиҰҒзҺҜиҠӮгҖӮиҫ“е…ҘеӣҫеғҸз”ұдәҺеӣҫеғҸйҮҮйӣҶзҺҜеўғзҡ„дёҚеҗҢпјҢеҰӮе…үз…§жҳҺжҡ—зЁӢеәҰд»ҘеҸҠи®ҫеӨҮжҖ§иғҪзҡ„дјҳеҠЈзӯүпјҢеҫҖеҫҖеӯҳеңЁжңүеҷӘеЈ°пјҢеҜ№жҜ”еәҰдёҚеӨҹзӯүзјәзӮ№гҖӮеҸҰеӨ–пјҢи·қзҰ»иҝңиҝ‘пјҢз„Ұи·қеӨ§е°ҸзӯүеҸҲдҪҝеҫ—дәәи„ёеңЁж•ҙе№…еӣҫеғҸдёӯй—ҙзҡ„еӨ§е°Ҹе’ҢдҪҚзҪ®дёҚзЎ®е®ҡгҖӮдёәдәҶдҝқиҜҒдәәи„ёеӣҫеғҸдёӯдәәи„ёеӨ§е°ҸпјҢдҪҚзҪ®д»ҘеҸҠдәәи„ёеӣҫеғҸиҙЁйҮҸзҡ„дёҖиҮҙжҖ§пјҢеҝ…йЎ»еҜ№еӣҫеғҸиҝӣиЎҢйў„еӨ„зҗҶгҖӮ
дәәи„ёеӣҫеғҸзҡ„йў„еӨ„зҗҶдё»иҰҒеҢ…жӢ¬дәәи„ёжү¶жӯЈпјҢдәәи„ёеӣҫеғҸзҡ„еўһејәпјҢд»ҘеҸҠеҪ’дёҖеҢ–зӯүе·ҘдҪңгҖӮдәәи„ёжү¶жӯЈжҳҜдёәдәҶеҫ—еҲ°дәәи„ёдҪҚзҪ®з«ҜжӯЈзҡ„дәәи„ёеӣҫеғҸпјӣеӣҫеғҸеўһејәжҳҜдёәдәҶж”№е–„дәәи„ёеӣҫеғҸзҡ„иҙЁйҮҸпјҢдёҚд»…еңЁи§Ҷи§үдёҠжӣҙеҠ жё…жҷ°еӣҫеғҸпјҢиҖҢдё”дҪҝеӣҫеғҸжӣҙеҲ©дәҺи®Ўз®—жңәзҡ„еӨ„зҗҶдёҺиҜҶеҲ«гҖӮеҪ’дёҖеҢ–е·ҘдҪңзҡ„зӣ®ж ҮжҳҜеҸ–еҫ—е°әеҜёдёҖиҮҙпјҢзҒ°еәҰеҸ–еҖјиҢғеӣҙзӣёеҗҢзҡ„ж ҮеҮҶеҢ–дәәи„ёеӣҫеғҸгҖӮдёӢйқўз®ҖеҚ•д»Ӣз»ҚдёҖдәӣйў„еӨ„зҗҶзҡ„ж–№жі•гҖӮ

вҖӢ еҜ№дәҺдәәи„ёжү¶жӯЈпјҢйҮҮз”Ёmatlab еҶ…зҪ®еҮҪж•°ginput,жүӢеҠЁзӮ№еҮ»дәәзҡ„дёӨзңјд»ҘеҸҠеҳҙе”ҮпјҢз„¶еҗҺиҮӘеҠЁиЈҒеҲҮдёә128*128зҡ„зҒ°еәҰеӣҫеғҸ

вҖӢ жңүдәӣз…§зүҮжӣқе…үиҝҮеәҰпјҢжҲ–иҖ…е…үжәҗдёҚи¶іпјҢдҫҝеҜ№е…¶иҝӣиЎҢеӨ„зҗҶпјҢеҲ©з”ЁimadjustпјҢеҜ№е…¶иҝӣиЎҢдә®еәҰи°ғжқ°гҖӮдҪҝеӣҫеғҸзҡ„з»ҶиҠӮжӣҙеҠ жё…жҘҡпјҢд»ҘеҮҸејұе…үзәҝе’Ңе…үз…§ејәеәҰзҡ„еҪұе“ҚгҖӮ
жңҖеҗҺimreshape еҜ№еӣҫеғҸзҹ©йҳөиҝӣиЎҢеҸҳеҪўеҺӢзј©пјҢimwrite дҪҝеӣҫеғҸдҝқеӯҳдёәbmpж јејҸгҖӮ

2 зү№еҫҒжҸҗеҸ–

еҜ№еӣҫеғҸжҸҗеҸ–Gaborзү№еҫҒпјҢе°әеәҰз»ҙж•°дёә5,и§’еәҰз»ҙж•°дёә8,дҪҝе…¶жҲҗдёә(6553601)зҡ„дёҖз»ҙеҲ—еҗ‘йҮҸ
и®Ўз®—64еј еӣҫзүҮзҡ„еқҮеҖјпјҢеҜ№ж•°жҚ®зҹ©йҳөпјҲ65536064пјүеҮҸеҺ»еқҮеҖјпјҢз„¶еҗҺи®Ўз®—еҚҸж–№е·®зҹ©йҳөгҖӮ
еӣ дёәжӯӨеҚҸж–№е·®зҹ©йҳөз»ҙж•°пјҲ655360*655360пјүиҝҮеӨ§пјҢиӢҘзӣҙжҺҘи®Ўз®—пјҢдјҡеј•еҸ‘out of memory ,йҮҮз”ЁдёҖдёӘе°ҸжҠҖе·§гҖӮ

еҒҮи®ҫ x дёә mГ—n зҡ„зҹ©йҳө, mвү«n,S=x*x^вҖҷ,PCAйңҖиҰҒи®Ўз®— S жңҖеӨ§зҡ„ k дёӘзү№еҫҒеҖјд»ҘеҸҠеҜ№еә”зҡ„зү№еҫҒеҗ‘йҮҸгҖӮдҪҶжҳҜmГ—mз»ҙеәҰеӨӘеӨ§пјҢж— жі•зӣҙжҺҘжұӮи§ЈгҖӮ

д»Ө C = x^вҖҷx ,CV=aV, a дёәзү№еҫҒеҖјпјҢVдёәзү№еҫҒеҗ‘йҮҸпјҢxCV=xx^вҖҷxV=axV, S(xV)=a*(xV).

жүҖд»ҘеҒҮи®ҫ a жҳҜ C зҡ„зү№еҫҒеҖјпјҢйӮЈд№Ҳ(a,xV) жҳҜSзҡ„зү№еҫҒеҖје’Ңзү№еҫҒеҗ‘йҮҸгҖӮеҰӮжӯӨеҸҜд»ҘеҢ–з®Җи®Ўз®—иҝҮзЁӢ
з„¶еҗҺи®Ўз®— жҜҸе№…еӣҫеңЁ еҚҸж–№е·®зҹ©йҳөиҙЎзҢ®зҺҮеүҚ95%зҡ„зү№еҫҒеҗ‘йҮҸжүҖжһ„жҲҗзҡ„еҹәдёҠзҡ„жҠ•еҪұгҖӮ

3. йў„жөӢеҲҶзұ»

ж №жҚ®зәҝжҖ§еҲҶзұ»еҷЁзҡ„йҳҗиҝ°пјҢи®Ўз®—е…¶жүҖеңЁеҲҶзұ»

4. еҸӮж•°и°ғж•ҙ

ж №жҚ®е®һйӘҢз»“жһңпјҢдҝ®ж”№Gaborе°әеәҰе’ҢдҪҚзҪ®еҸӮж•°пјҢеҜ»жұӮжңҖдҪіеҸӮж•°гҖӮ

еӣӣгҖҒ е®һйӘҢз»“жһң

е…¶дёӯе®һйӘҢдёҖдёәпјҢдёҚеҜ№еӣҫеғҸиҝӣиЎҢд»»дҪ•еӨ„зҗҶпјҢд»…иҪ¬дёә128128зҡ„bmpеӣҫеғҸпјҢз„¶еҗҺиҝӣиЎҢе®һйӘҢгҖӮ
е®һйӘҢдәҢдёәпјҢеҜ№еӣҫеғҸиҝӣиЎҢиЈҒеҲҮж“ҚдҪңпјҢиҪ¬дёә128128зҡ„bmpеӣҫеғҸ,дҪҝеӣҫеғҸдё»дҪ“дёәйқўйғЁпјҢдё”еҜ№еӣҫеғҸиҝӣиЎҢдә®еәҰи°ғиҠӮгҖӮ
е®һйӘҢдёүдёәпјҢеҜ№еӣҫеғҸиҝӣиЎҢиЈҒеҲҮпјҢдё”еҜ№еӣҫеғҸиҝӣиЎҢдә®еәҰи°ғиҠӮпјҢеҲ©з”ЁGaborе°ҸжіўжҸҗеҸ–дә”е°әеәҰе…«ж–№еҗ‘зү№еҫҒпјҢеңЁдёҚеҗҢеҸӮж•°пјҢдёҚеҗҢжөӢиҜ•йӣҶдёӢзҡ„е№іеқҮеҮҶзЎ®зҺҮ
е®һйӘҢеӣӣ еҜ№еӣҫеғҸиҝӣиЎҢиЈҒеҲҮпјҢдё”еҜ№еӣҫеғҸиҝӣиЎҢдә®еәҰи°ғиҠӮпјҢеҲ©з”ЁGaborе°ҸжіўжҸҗеҸ–дә”е°әеәҰе…«ж–№еҗ‘зү№еҫҒпјҢе…¶дёӯе°әеәҰгҖҒж–№еҗ‘зү№еҫҒйҖүеҸ–дёәеӯҰд№ еҫ—еҲ°зҡ„жңҖдјҳеҸӮж•°гҖӮ
йҷӨе®һйӘҢдёүеӨ–пјҢе…¶дҪҷе®һйӘҢеқҮдёә16дёӘжөӢиҜ•йӣҶеҫ—еҲ°зҡ„дёҖж¬ЎеҮҶзЎ®зҺҮгҖӮ

е°Ҷ20дёӘдәәеҲҶдёә20дёӘзұ»еҲ«пјҢеҸҜд»ҘеҮҶзЎ®йў„жөӢ15дёӘдәәзҡ„иә«д»ҪпјҢе…¶дёӯз¬¬5дёӘдәәйў„жөӢеӨұиҙҘпјҢеҮҶзЎ®зҺҮиҫҫеҲ° 70.00%

е®һйӘҢз»“жһңи®Ёи®ә пјҡ

1.еҲ©з”ЁPCAеҸҜд»ҘеҜ№ж•°жҚ®иҝӣиЎҢйҷҚз»ҙеҺӢзј©пјҢдҫҝдәҺд№ӢеҗҺзҡ„ж“ҚдҪңеӨ„зҗҶ
2.еҺҹеӣҫдёӯжңүи®ёеӨҡеҷӘйҹіжҲ–дёҺдәәи„ёзү№еҫҒж— е…ізҡ„иғҢжҷҜгҖҒзҒҜе…үзӯүпјҢеҜ№еӣҫеғҸиҝӣиЎҢеӨ„зҗҶеҗҺпјҢиҜҶеҲ«еҮҶзЎ®зҺҮеҫ—еҲ°жҸҗеҚҮгҖӮ
3.иӢҘзӣҙжҺҘиҝҗз”ЁPCAз®—жі•иҝӣиЎҢиҜҶеҲ«пјҢеҸҜд»ҘжҺЁеҮәпјҢжӯӨз®—жі•еҜ№дәәи„ёзҡ„ж–№еҗ‘гҖҒе…үз…§зҡ„ејәеәҰгҖҒдәәи„ёдҫ§йқўжҲ–жҳҜжӯЈйқўе…·жңүиҫғејәзҡ„ж•Ҹж„ҹжҖ§пјҢиҖҢGaborзү№еҫҒеҸӘиғҪи§ЈеҶіе…үз…§ж•Ҹж„ҹжҖ§гҖҒд»ҘеҸҠеӣҫеғҸж–№еҗ‘жҖ§иҝҷдёӨдёӘй—®йўҳгҖӮеҜ№е®һйӘҢз»“жһңеҸҜзҹҘпјҢиҜҶеҲ«еӨұиҙҘзҡ„дәәи„ёзұ»еҲ«дё»иҰҒдёәеңЁи®ӯз»ғйӣҶжҲ–и®ӯз»ғйӣҶдёӯеҮәзҺ°дҫ§и„ёжӢҚз…§зҡ„жғ…еҶөгҖӮиӢҘж”№иҝӣиҝҷдёӘй—®йўҳпјҢдәәи„ёиҜҶеҲ«зҡ„еҮҶзЎ®зҺҮе°Ҷдјҡеҫ—еҲ°иҝӣдёҖжӯҘзҡ„жҸҗеҚҮгҖӮеҰӮдҪ•ж”№иҝӣиҝҷдёӘй—®йўҳпјҢеҲҷжҳҜе®һйӘҢеҸҜд»ҘиҝӣдёҖжӯҘе®Ңе–„зҡ„ең°ж–№
